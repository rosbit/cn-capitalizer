## 数字转为中文大写

一遍扫描，全程无替换操作。

 1. 金额
    - 可以转换到`万万亿`级别
    - 如有需要，可以在单位中追加更高级别
 1. 年份
    - 2021可以转为“贰零贰壹”或“贰仟零贰拾壹”
 1. 月份
    - 支持支票规范的月份，如10月转为“零拾”月
    - 普通月份支持，如10月转为“拾”月
 1. 日纷
    - 支持支票规范的日份，如20日转为“零贰拾”日
    - 普通日份支持，如20日转为“廿”日
